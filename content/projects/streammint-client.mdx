Streammint Client is the TypeScript-based frontend for the Streammint platform. It provides the user-facing interface for interacting with real-time streams, managing media, and coordinating with the Rust-powered backend.

Built with modern frontend tooling, the client focuses on responsiveness, clear UX, and tight integration with Streammint Server APIs.

### Key Features

- **Streaming UI** – Interfaces for viewing and interacting with live or on-demand streams  
- **Media Management** – Controls for browsing, filtering, and managing media content  
- **API Integration** – Typed integration with the Streammint Server for stable contracts  
- **Modern Frontend Stack** – Built with TypeScript and a component-driven architecture  
- **Real-Time Feedback** – Designed around live updates and responsive interactions
