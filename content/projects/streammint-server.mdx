Streammint Server is a pluggable Rust backend for real-time chat. It’s designed to be a solid foundation for modern messaging experiences, letting you plug in custom logic, storage, and transports without rewriting the entire system.

With a focus on extensibility and performance, Streammint Server helps you build chat backends that are easy to evolve as your product grows.

### Key Features

- **Pluggable Architecture** – Swap or extend modules for storage, auth, or transports  
- **Real-Time Messaging** – Built for low-latency chat and live conversations  
- **Rust Core** – Leverages Rust for performance and reliability in production workloads  
- **Clean APIs** – Provides a clear contract for clients (like Streammint Client) and services  
- **Scalable Design** – Structured to support multiple chat experiences and deployment setups
