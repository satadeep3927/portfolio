Loom-UI is the monitoring and observability dashboard for the Loom workflow engine. It provides a visual interface to inspect workflows, track executions, and understand the health of your long-running processes.

Built with TypeScript and modern frontend patterns, Loom-UI turns Loomâ€™s event-sourced data into an accessible, operator-friendly view of your system.

### Key Features

- **Workflow Overview**: Visualize active, completed, and failed workflows at a glance
- **Execution Details**: Inspect individual workflow runs, events, and state transitions
- **Status & Health**: Monitor system health, failures, and bottlenecks
- **Search & Filtering**: Quickly find workflows by status, ID, or other attributes
- **TypeScript-First**: Strongly-typed components and data models for reliability
- **Extensible UI**: Designed to grow with new Loom features and metrics
