Developed and maintained ATXFloods, a critical web application designed to monitor flood conditions and road closures across Texas State. Leveraging Python and Django for robust backend services and React for a dynamic and responsive frontend, the application provides real-time updates and alerts to improve public safety and aid in disaster management.

### Key Features
- Real-time flood monitoring and road closure alerts
- Scalable architecture handling peak usage during emergencies
- Integration with multiple data sources for comprehensive coverage
- High availability and reliability for critical emergency response
- Public safety dashboard with real-time updates
- Cross-platform compatibility (Windows, macOS, Linux)
